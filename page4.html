# css操作

取值

$("p").css("color");

设值

$("p").css("color","red");

同时设置多个值

$("p").css({ 
    "color": "#ff0011",
    "background": "blue"
});

# 位置

  • offset()

获取匹配元素在当前视口的相对偏移。

var offset = $("p").offset();
console.log( "left: " + offset.left );
console.log( "top: " + offset.top );
  • scrollTop()

获取第一段相对滚动条顶部的偏移

console.log( $("p").scrollTop() );

# 尺寸

  • height([val]) 取得匹配元素当前计算的高度值
  • width([val]) 取得匹配元素当前计算的宽度值
