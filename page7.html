# 效果

# 基本

  • show([s,[e],[fn]]) 显示效果
  • hide([s,[e],[fn]]) 隐藏效果
  • toggle([s],[e],[fn]) 切换显隐

# 滑动

  • slideDown([s],[e],[fn]) 向下滑动
  • slideUp([s,[e],[fn]]) 向上滑动
  • slideToggle([s],[e],[fn]) 切换滑动

# 淡入淡出

  • fadeIn([s],[e],[fn]) 淡入
  • fadeOut([s],[e],[fn]) 淡出
  • fadeTo([[s],o,[e],[fn]]) 淡出指定透明度
  • fadeToggle([s,[e],[fn]]) 切换淡入淡出

# 自定义

  • animate(p,[s],[e],[fn]) 自定义动画
  • stop([c],[j]) 停止动画过渡
  • delay(d,[q]) 延时
