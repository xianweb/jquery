# 文档处理

# 内部插入

  • append(content|fn) 内部后面出入，返回父元素
  • appendTo(content) 内部后面出入，返回被插入元素
  • prepend(content|fn) 内部前面出入，返回父元素
  • prependTo(content) 内部前面出入，返回被插入元素

# 外部插入

  • after(content|fn) 外部后面出入，返回父元素
  • before(content|fn) 外部后面出入，返回被插入元素
  • insertAfter(content) 外部后面出入，返回父元素
  • insertBefore(content) 外部前面出入，返回被插入元素

# 包裹

  • wrap(html|ele|fn) 把所有匹配的元素用其他元素的结构化标记包裹起来。
  • unwrap() 卸载包裹
  • wrapAll(html|ele) 将所有匹配的元素用单个元素包裹起来
  • wrapInner(html|ele|fn)

# 删除

  • empty() 清空
  • remove([expr]) 删除

# 复制

  • clone([Even[,deepEven]])
