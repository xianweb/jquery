# 选择器

# 基本

  • #id 选择器
  • element 标签选择器
  • class 类选择器
  • * 匹配所有
  • selector1, selector2, selectorN 多条件匹配
  • [attribute] 匹配包含某属性
  • [attribute=value] 匹配包含某属性并且值也被指定

# 层级

  • ancestordescendant 目标元素 子孙元素
  • parent > child 目标元素 子元素
  • prev + next 目标元素紧挨着下一个元素
  • prev ~ siblings 目标元素紧挨着所有元素

# 筛选器

  • :first 匹配第一个
  • :not(selector) 匹配不包含
  • :even 匹配偶数
  • :odd 匹配奇数
  • :eq(index) 匹配指定索引值
  • :gt(index) 匹配大于指定索引值
  • :last 匹配最后一个
  • :lt(index) 匹配小于指定索引值

# 内容

:contains(text) 匹配包含某文字 :empty 匹配所有不包含子元素或者文本的空元素 :has(selector) 匹配含有选择器所匹配的元素的元素
