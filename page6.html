# 页面载入

ready(fn)

在DOM加载完成时运行的代码，可以这样写：

$(document).ready(function(){
  // 在这里写你的代码...
});

//缩写
$(function(){

})

#window.onload 和 ready(fn) 区别：

# 执行次数不同

  1. window.onload：只能执行一次，如果第二次，那么第一次的执行会被覆盖

  2. ready(fn)：可以执行多次，第N次都不会被上一次覆盖

# 执行时间点不同

  1. window.onload：必须等待网页全部加载完毕（包括图片等），然后再执行JS代码

  2. ready(fn)：可以执行多次，第N次都不会被上一次覆盖

# 事件处理

  • on(eve,[sel],[data],fn)
$("p").on("click", function(){
    alert( $(this).text() );
});

//传参到回调函数内
$("p").on("click", {foo: "bar"}, function(){
    alert(event.data.foo);
});
  • off(eve,[sel],[fn])

在选择元素上移除一个或多个事件的事件处理函数。

//移除所有事件
$("p").off();
  • one(type,[data],fn)

为每一个匹配元素的特定事件（像click）绑定一个一次性的事件处理函数。

$("p").one("click", function(){
  alert( $(this).text() );
});
  • trigger(type,[data])

在每一个匹配的元素上触发某类事件。

$("form:first").trigger("submit");

# 事件代理（委托）

$("ul").on("click", 'li', function(){
    console.log(this);
});

# 事件切换

  • hover([over,]out)
$("td").hover(
  function () {
    $(this).addClass("hover");
  },
  function () {
    $(this).removeClass("hover");
  }
);
